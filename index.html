Embroidery Types

Hand Work & Machine Work

01
Sozni Embroidery
Fine needle work done entirely by hand. Intricate floral and paisley patterns. Takes 3–18 months per shawl. Highest value category.
02
Aari (Crewel) Work
Hook-needle embroidery with thick woollen threads. Dense, colourful patterns. Used on suits and shawls both.
03
Tilla (Zari) Work
Gold and silver metallic thread embroidery. Used for bridal and festive pieces. Machine-assisted options available at lower price points.
04
Kani Weave
Pattern woven directly into the shawl during weaving. No embroidery added after. The oldest Kashmiri weaving tradition.
05
Machine Embroidery
Computer-aided patterns on shawls and suits. Faster production, affordable pricing. Ideal for bulk orders and gifting.
06
Block / Papier-Mâché Print
Traditional block printing on suit fabric. Unique patterns inspired by Kashmiri papier-mâché art, not found in machine-made textiles.
Our Story

From the Hands of Kashmir to Your Doorstep

In the narrow lanes of old Srinagar, behind wooden lattice windows, master artisans sit cross-legged with needles finer than a strand of hair. They embroider patterns passed down through generations — each stitch a quiet act of devotion to a craft that has survived centuries.

We work directly with these artisans. No showrooms. No middlemen. No markups. When you buy from us, you're not buying from a warehouse — you're buying from the hands that made it. And we deliver every piece with the same care and pride the artisan put into crafting it.

    One-time setup — takes 10 minutes

    Step 1. Go to sheets.google.com and create a new sheet.

    Step 2. Name row 1 headers exactly:
    Order ID | Date | Customer Name | Phone | Email | Address | Items | Total | Payment | Status

    Step 3. Click Extensions → Apps Script. Delete existing code. Paste this:

    var SECRET_TOKEN = 'change-this-secret-token';

function doPost(e) {
  var data = JSON.parse(e.postData.contents);
  if (data.token !== SECRET_TOKEN) {
    return ContentService.createTextOutput(
      JSON.stringify({result:'unauthorised'})
    ).setMimeType(ContentService.MimeType.JSON);
  }
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  sheet.appendRow([data.orderId, data.date, data.name, data.phone,
    data.email, data.address, data.items, data.total,
    data.payment, data.status]);
  return ContentService.createTextOutput(
    JSON.stringify({result:'success'})
  ).setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  return ContentService.createTextOutput(
    JSON.stringify({status:'active'})
  ).setMimeType(ContentService.MimeType.JSON);
}

    ⚠ Change change-this-secret-token to the same value you set in Settings above.

    Step 4. Click Deploy → New Deployment → Web App → Anyone → Deploy.

    Step 5. Copy the Web App URL. Paste it in the field above. Save Settings.

    ✅ Every order will now log automatically to your Google Sheet.
